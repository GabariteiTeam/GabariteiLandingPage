O Gabaritei é um framework pensado especialmente para o desenvolvimento de ambientes virtuais de ensino e aprendizagem para o uso em escolas e cursinhos.

Saiba mais

Nós temos o que você precisa!

O Gabaritei é um framework de código livre, flexível e de fácil uso, que utiliza tecnologias atuais tais como Ruby on Rails e o AngularJS. Além disso, o framework pode ser facilmente modificado e adaptado, seguindo um processo que permite adicionar novas funções sem prejudicar a estrutura sólida criada inicialmente através da Engenharia de Software.
